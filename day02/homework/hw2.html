3# ახსენით კომენტარის სახით რა განსხვევებაა (HTML, CSS და JavaScript) შორის

HTML არის ვებსაიტის მონაცემთა სტრუქტურა, როგორიცაა მაგალითად ტექსტი, სურათები, ვიდეოები, სათაურები და ა.შ. CSS არის ვებსაიტის ვიზუალური სტილი, როგორიცაა ფერები, შრიფტები, განლაგება და სხვა ვიზუალური ნაწილები. JS არის ვებსაიტის ფუნქციონალობა, რომელიც საშუალებას იძლევა ვებსაიტის ინტერაქციულობაში, ე.წ. დინამიურობაში.
